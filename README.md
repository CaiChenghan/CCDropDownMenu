## Installation

CCDropDownMenu is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'CCDropDownMenu', git: 'https://github.com/CaiChenghan/CCDropDownMenu.git'
```
